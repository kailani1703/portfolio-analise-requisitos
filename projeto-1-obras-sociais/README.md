# Projeto 1 — Sistema de Gestão de Turmas
## Obras Sociais Francisco Thiesen

### Descrição
Sistema para gestão de turmas, alunos e presenças de uma ONG de ensino de lógica de programação para adolescentes.

### Entregáveis
- Documento de Requisitos completo
- 5 Histórias de Usuário com Critérios de Aceitação
- Backlog priorizado com MoSCoW no Jira
- Diagrama BPMN (draw.io)
- Fluxo de Usuário (Miro)

### Ferramentas utilizadas
Jira · draw.io · Miro

### Stakeholders
- Coordenador
- Instrutor
- Aluno
